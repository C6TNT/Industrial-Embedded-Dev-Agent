# Data Directory

This directory contains the first normalized project assets for the MVP stage.

## Layout

- `benchmark/`
  Benchmark seeds in JSONL format, including the original combined file and split subsets.
- `taxonomy/`
  Tag definitions and category notes used to constrain structured outputs.
- `materials/`
  Material indexes and seed asset inventories for documents, logs, cases, and scripts.

## Notes

- The current content is seeded from `准备产物/`.
- This directory is intended to become the stable interface between raw project notes and later retrieval, evaluation, and agent workflows.
- Version `v1` assets are still human-curated and should be treated as baseline seeds instead of final production datasets.
