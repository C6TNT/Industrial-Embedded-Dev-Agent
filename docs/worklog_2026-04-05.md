## 九、今日新增的 compare-pack

在今天最后一轮，还继续把双包差异比较能力补上了。

新增命令：

- `ieda tools compare-pack <left.json> <right.json>`

这条命令会自动比较两次只读快照的差异，重点看：

- request 是否变化
- session 是否一致
- tool / risk / mode 是否变化
- returncode / parsed status / transport_state 是否变化
- `axis0 / axis1` 的 `error_code / status_code / axis_status / encoder` 是否变化

并且默认会输出一份 Markdown 差异报告，方便直接放进联调记录或 issue 追踪。
