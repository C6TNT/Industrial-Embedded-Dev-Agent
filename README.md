# Industrial Embedded Dev Agent

Industrial Embedded Dev Agent is an open-source project workspace for building an industrial-flavored embedded troubleshooting and engineering assistant.

The current repository focuses on an i.MX8MP based industrial control scenario and keeps the materials that are most useful for:

- embedded debug knowledge organization
- log analysis and attribution
- industrial issue case accumulation
- tool safety benchmark design
- agent evaluation and future MVP implementation

## Repository Layout

- `Industrial Embedded Dev Agent_项目方案.md`: project plan and scope definition
- `Industrial Embedded Dev Agent_项目评估.txt`: external evaluation reference entry
- `准备产物/`: prepared benchmark, taxonomy, and seed materials
- `资料/`: retained public-facing engineering references and code samples
- `仓库整理说明.md`: repository cleanup policy and large-file strategy

## Open Source Scope

This repository intentionally excludes oversized local reference bundles and demo packages that are better kept outside source control.

Examples of excluded local-only materials:

- vendor demo bundles
- oversized PDF manuals
- board bring-up raw handbooks
- duplicated intermediate workspaces

See `仓库整理说明.md` for the current policy.

## Current Status

The project is in the preparation and repository-hardening stage. The next major step is to build the first MVP agent workflow on top of:

- curated industrial materials
- risk-aware labels
- benchmark-driven evaluation
